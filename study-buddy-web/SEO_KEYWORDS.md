## Keyword Density Guidelines

- **Primary keyword (Study Buddy)**: 1-2% density
- **Long-tail keywords**: Use naturally, 2-3 times per 1000 words
- **Related terms**: Use synonyms and variations
- **Avoid keyword stuffing**: Write for humans first, SEO second

## SEO Best Practices

1. **Use keywords in:**
   - Blog post titles (H1)
   - First paragraph
   - Subheadings (H2, H3)
   - Image alt text
   - Meta descriptions
   - URL slugs

2. **Internal linking:**
   - Link to Study Buddy features
   - Link to related blog posts
   - Use descriptive anchor text

3. **External linking:**
   - Link to authoritative sources
   - Cite studies and research
   - Link to educational resources

4. **Content length:**
   - Aim for 1,500-2,500 words for pillar content
   - 800-1,200 words for supporting articles
   - Include images, videos, and interactive elements
